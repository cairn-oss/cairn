# Cairn v0.1.0 — the local-first cost + security wedge

**The first public release.** Cairn scans Terraform on *your* machine,
ranks security, cost, reliability and governance findings in a single
pass, reconciles the trade-offs between them, and proposes concrete fixes.
Nothing leaves your environment.

## Highlights

- **Cost + security in one pass.** A change that saves money can widen your
  attack surface — Cairn is the tool that tells you at the moment of the
  decision, and sequences the fixes when both hit the same resource.
- **Fix, not just flag.** Every finding carries plain-English remediation
  and, where safe, a ready-to-apply HCL patch.
- **Local-first by architecture.** No SaaS, no upload, no telemetry. LLM
  explanations are strictly opt-in and bring-your-own-key (or fully local
  via Ollama).
- **Read-only, audited (Trust Ladder rung 0).** Cairn never touches your
  files, cloud, or git state; every scan is recorded to a local
  append-only log.

## What's included

- 18 rules across security (SEC001–SEC009), cost (COST001–COST005),
  reliability (REL001–REL002) and governance (GOV001–GOV002)
- Policy-as-code (`.cairn.yaml`): severity floors, CI fail thresholds,
  required tags, disabled rules, severity overrides, glob ignores
- Reporters: console, JSON (versioned schema), SARIF 2.1.0 (GitHub Code
  Scanning), Markdown
- Stable CI exit-code contract (0 clean / 1 findings / 2 error)
- Distribution: PyPI (`cairn-iac`), non-root Docker image, composite
  GitHub Action

## Install

```bash
pip install cairn-iac==0.1.0
cairn scan ./infra
```

## Cloud coverage note

This release is **not yet fully cloud-agnostic**. Its rule packs cover AWS
only.
Terraform for other providers (Azure, GCP, Oracle, VMware
vSphere, ...) is *parsed* but not yet *checked*. Broader multi-cloud and
on-prem coverage arrives in later releases (Azure/GCP/vSphere packs in
v0.4.5; coverage-transparency reporting in v0.5.1, which makes unscanned
resources explicit rather than reporting a misleading "clean"). See
[docs/coverage.md](../docs/coverage.md).
